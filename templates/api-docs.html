组件数据获取 POST

获取指定组件的数据，支持自定义查询参数和过滤条件。所有数据请求统一使用POST方式，提供更灵活的参数传递。

POST /api/components/{component_id}/data

请求参数

参数名 类型 必填 描述
component_id string 是 组件ID，如：line_basic, bar_basic, pie_basic
dateRange object 否 日期范围过滤，包含start和end字段
dimensions array 否 数据维度选择，如：["region", "product"]
filters object 否 自定义过滤条件
aggregation string 否 数据聚合方式，如：sum, avg, count
limit number 否 返回数据的数量限制

请求示例

// 基础请求 - 获取柱状图数据 POST /api/components/bar_basic/data Content-Type: application/json { "dateRange": { "start": "2024-01-01", "end": "2024-12-31" }, "dimensions": ["region", "product"], "filters": { "status": "active", "category": "electronics" }, "aggregation": "sum", "limit": 100 }

JavaScript 调用示例

// 使用 fetch API 调用 async function getComponentData(componentId, params = {}) { try { const response = await fetch(`/api/components/${componentId}/data`, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify(params) }); if (!response.ok) { throw new Error(`HTTP error! status: ${response.status}`); } const result = await response.json(); return result; } catch (error) { console.error('获取组件数据失败:', error); throw error; } } // 调用示例 const chartData = await getComponentData('bar_basic', { dateRange: { start: '2024-01-01', end: '2024-12-31' }, filters: { region: '华东' } });

响应格式

{ "success": true, "data": { "title": { "text": "基础柱状图", "left": "center" }, "tooltip": { "trigger": "axis" }, "legend": { "data": ["销量"] }, "xAxis": { "type": "category", "data": ["周一", "周二", "周三", "周四", "周五", "周六", "周日"] }, "yAxis": { "type": "value" }, "series": [{ "name": "销量", "type": "bar", "data": [120, 200, 150, 80, 70, 110, 130] }] }, "timestamp": "2024-01-15T10:30:00.000Z", "component_id": "bar_basic", "request_params": { /* 请求参数回显 */ } }

支持的组件类型

系统支持以下组件类型，每种组件都有对应的数据格式和配置选项。

组件ID 组件名称 分类 数据格式
line_basic 基础折线图 ECharts图表 ECharts Option
bar_basic 基础柱状图 ECharts图表 ECharts Option
pie_basic 基础饼图 ECharts图表 ECharts Option
scatter_basic 基础散点图 ECharts图表 ECharts Option
radar_basic 基础雷达图 ECharts图表 ECharts Option
gauge_basic 基础仪表盘 ECharts图表 ECharts Option
text_basic 文本框 基础组件 Text Properties
image_basic 图片组件 基础组件 Image Properties
datetime_basic 时间控件 基础组件 DateTime Properties

错误代码说明

状态码 错误类型 描述 解决方案
200 成功 请求处理成功 -
400 请求错误 请求参数格式错误或缺少必要参数 检查请求参数格式和必填字段
404 组件未找到 指定的组件ID不存在 检查组件ID是否正确
500 服务器错误 服务器内部错误 联系系统管理员
